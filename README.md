# PoE-Gem-Flipping
Calculate expected value of PoE's gem flipping

## Getting started
1. Clone repository
    ```bash
    git clone https://github.com/yehoon17/PoE-Gem-Flipping.git
    ```

2. Install requirements
    ```bash
    pip install -r requirements.txt
    ```

3. Run server
    ```bash
    flask --app main run
    ```
